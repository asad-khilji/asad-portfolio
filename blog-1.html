HTML and CSS

March 4, 2025

HTML or Hyper Text Markup Language

The HTML or Hyper Text Markup Language is a front end programming language used to structure the web page.

HTML is displayed as a client side programming language.

Client side programming languages are languages that are visible from the browser, where the user can see.

Building an html page is very simple

An HTML document consists of elements and tags to structure the web page.

CSS or Cascading Style Sheets

CSS or Cascading Style Sheets is a prograaming language used to add styling to the webpage.

CSS is a front end programming language.

CSS is displayed as a client side programming language.

Client side programming languages are visible on the browser, where the user can see.

CSS consists of selectors that link to the elements and tags of the html page.

There are many different ways to insert CSS onto the webpage.

There is in style CSS within the HTML page.

There is separate file where CSS styling is added then linked to html page.

There is in style within the page and code of each element of the html tag.
